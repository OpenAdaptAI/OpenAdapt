# Noncanonical historical documentation

This directory is retained only to avoid breaking old repository links. It is
**not** the source for [docs.openadapt.ai](https://docs.openadapt.ai) and must
not deploy to that domain.

The canonical documentation source and deployment pipeline live in
`OpenAdaptAI/openadapt-ops` (the local `openadapt-maintenance` checkout). That
site documents the current demonstration compiler, governed replay, product
maturity, hosted boundaries, and known limits.

Start here:

- [Current product documentation](https://docs.openadapt.ai)
- [Canonical engine](https://github.com/OpenAdaptAI/openadapt-flow)
- [Launcher and migration guidance](https://github.com/OpenAdaptAI/OpenAdapt#readme)

The remaining files under this directory describe earlier package/training
architectures. They are historical reference, not current onboarding or product
claims.
